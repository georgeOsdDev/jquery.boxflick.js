jQuery.boxflick.js

Enable horizontal flick move for Android

jquery.boxflick.js achieve the scrolling with the flick of a horizontal direction in Android devices, by using the transform3d of css3.
This plugin have been tested and approved for use only in the webkit-browser.
If there is a bug, enjoy hacking.

* html(style)

<div id="box" style="width:400px;height:100px;overflow:hidden;">
  <div id="handler" style="float:left; width:2000px;height:100px; left:0px;">
    <div style="float:left;width:400px;height:100px;"></div>
    <div style="float:left;width:400px;height:100px;"></div>
    <div style="float:left;width:400px;height:100px;"></div>
    <div style="float:left;width:400px;height:100px;"></div>
    <div style="float:left;width:400px;height:100px;"></div>
  </div>
  <div style="clear:left;"></div>
</div>
Note: <div id="box"> is not always necessary.

* javascript

<script src="//cdnjs.cloudflare.com/ajax/libs/jquery/2.0.0/jquery.min.js"></script>
<script type="text/javascript">
  window.jQuery || document.write('<script src="jquery.min.js"><\/script>')
</script>
<script src="jquery.boxflick.js"></script>
<script type="text/javascript">
  $(function(){
    $("#handler").boxflick({
      auto:5000
    });
  })
</script>

* options

  • Number bonceLeft : specify length of bounce animation. (default:50)
  • Number bonceRight : specify length of bounce animation. (default:50)
  • String bounceLeftClass : specify class name when box reached to the start position. (default:undefined)
  • String bounceRightClass : specify class name when box reached to the end position. (default:undefined)
  • Number auto : specify time for auto animation.(default:0)
