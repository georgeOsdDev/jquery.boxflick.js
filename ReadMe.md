# jquery.boxflick.js
Enable horizontal flick move for Android.

## Usage

	<script src="jquery.min.js"></script>
	<script src="jquery.boxflick.js"></script>
	<script type="text/javascript">
			$(function(){
				$( "#handler" ).boxflick({
		 			auto:5000
				});
			});
	</script>

>See also > [Demo](http://georgeosddev.github.io/jquery.boxflick.js/)
